title: Simple Help Site 

This is a simple help page consisting only of a single page.
 
It's based on **A Little Hypertext System** so it includes:
  * Hyperlinks to other help pages
  * Simple searching ability
  * History
  * Simple wiki formatting
  * and many other features

Here follows an example  for an indentation ...

> - item 1
  - item 2
  - item 3
    -- subitem 3.1
    -- subitem 3.2
  -item 4

Here an example for an code block

   set x 5
   puts \$x
